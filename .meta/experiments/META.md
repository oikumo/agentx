# Experiments

> **Purpose**: Test new libraries, prototype features
> **Target**: AI agents
> **Rule**: Document and integrate or clean up

---

## Rules

**DO**: Date folders, document findings, clean up
**DON'T**: Leave broken, mix with production, skip documentation

## Lifecycle

1. **CREATE**: Clear hypothesis/goal
2. **DOCUMENT**: What + why
3. **VALIDATE**: Test hypothesis
4. **DECIDE**: Integrate / discard
5. **CLEAN**: Remove or archive
