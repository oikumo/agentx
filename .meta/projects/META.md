# Projects

> **Purpose**: Project plans and design documents
> **Target**: AI agents and developers
> **Rule**: One folder per project, contains design/plan docs

---

## Project structure
**Project ID**: Unique number with the prefix `P` and suffix `.`, ex: `P1-`
**Project Folder** : `P<ID>.<project-name>`
**Project Description** : `P<ID>.<project-name>/PROJECT.md`
**Project Plan** : `P<ID>.<project-name>/PLAN.md`
**Complementary Files Folders** `P<ID>.<project-name>/<complement-name>/`


## Rules

**DO**: Folder per project, document scope, link to experiments
**DON'T**: Include code, mix unrelated projects, leave outdated
