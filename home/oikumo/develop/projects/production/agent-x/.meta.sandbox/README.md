# Agent‑X Overview

Agent‑X is a modular AI development harness. It exposes a **safe sandbox** (`.meta.sandbox/`) for code experiments, a **testing ground** (`.meta.tests_sandbox/`) for TDD, an **experiments pool** (`.meta.experiments/`) for prototypes, and a **tool repository** (`.meta.development_tools/`) for custom extensions.

Core directives are preserved, and all interactions should follow the path hierarchy and token budgets defined in this harness.
