## Usage Guide

### When creating a new META.md:

1. Copy this template
2. Replace all bracketed placeholders `[...]`
3. Remove unused sections
4. Add specific details for your directory
5. Ensure word count is 100-500 words
6. Include at least one workflow example
7. Add quality standards
8. Review against checklist

### Quality Checklist

Before finalizing:

- [ ] Purpose is clear and concise
- [ ] Target audience is defined
- [ ] At least 3 DO rules
- [ ] At least 3 DON'T rules
- [ ] At least one workflow documented
- [ ] At least one example provided
- [ ] Structure diagram included
- [ ] Word count 100-500
- [ ] Related directories listed
- [ ] Version information current

### Common Mistakes to Avoid

1. **Too vague**: Be specific about purpose and rules
2. **Too long**: Keep it concise (100-500 words)
3. **No examples**: Always include concrete examples
4. **No workflows**: Show how to use the directory
5. **Outdated**: Keep version info current
6. **Isolated**: Link to related directories
