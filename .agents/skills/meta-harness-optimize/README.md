# Meta Harness Optimize

Comprehensive optimization of Meta Project Harness documentation, structure, and workflows. Achieves 30-50% token reduction while improving clarity and agent efficiency.

## Quick Start

```bash
skill meta-harness-optimize
```

**Time**: 30 min (token audit), 1-2 hrs (full optimization)
**Savings**: 30-50% token reduction

## What This Skill Does

Combines token optimization with structure and workflow analysis to create a comprehensive optimization system:

1. **Token Optimization** - Reduces token consumption by 30-50%
2. **Structure Analysis** - Ensures clear, logical directory organization
3. **Workflow Enhancement** - Streamlines agent workflows
4. **Continuous Monitoring** - Ongoing optimization with metrics tracking

## Core Workflows

### 1. Token Audit (30 min)
Baseline measurement and cost optimization

### 2. Documentation Compression (45 min)
Compress docs while preserving information using:
- Redundancy removal (75% savings)
- Table conversion (47% savings)
- Language simplification (80% savings)
- Decision trees (75% savings)

### 3. Structure Optimization (1-2 hrs)
Reorganize for clarity and navigation

### 4. Workflow Enhancement (1-2 hrs)
Streamline verbose workflows

### 5. Continuous Monitoring
Ongoing optimization with scheduled reviews
